# Project: Customer Relationship Management - CRM
### by Yashwanth Manchikatla

## Description
A CRM(Customer relationship management) system created using Spring MVC and Hibernate. It supports all the CRUD operations

## Functionality
- List Customers
- Add a new customer
- Update a customer
- Delete a customer
